Rekrutierung Box für ilchClan 1.1:
----------------------------------

Beschreibung:
-------------
Dieses Box-Modul zeigt euch in einer Box die WoW Klassen mit Ihren Specs an.
Wenn Ihr Backend eingestellt habt das Ihr die Klasse mit den Spec sucht wird das Spec Icon in der Box deutlicher dargestellt.

Entwickelt
----------
° von "Siggi"
 
Installation:
-------------
° alle Dateien im Ordner upload, in ihrer Ordnerstruktur hochladen
° Nach uploaden aller Datein muss die wowrecruit_install.php ausgeführt werden.
    => index.php?wowrecruit_install
° Im Adminbereich unter Navigation die Box recruit.php einbinden.
° Anschließend im Adminbereich unter Rekrutierung eure Klassen suche anpassen.
